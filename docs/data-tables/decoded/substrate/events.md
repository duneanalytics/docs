# Events

## **{chain}.events**

This table contains historical events information:

| **Column name**       | **Type**                        | **Description**                                          |
| --------------------- | ------------------------------- | -------------------------------------------------------- |
| block_time        | timestamp   |
| block_number      | bigint      |
| extrinsic_id      | string      |
| event_id          | string      |
| data              | string      |
| extrinsic_hash    | binary      |
| block_hash        | binary      |
| data_decoded      | string      |
